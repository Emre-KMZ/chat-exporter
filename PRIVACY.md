# Privacy Policy

**Chat Exporter** is a Chrome extension that exports conversations from AI chat platforms as local files.

## Data Collection

Chat Exporter does not collect, store, transmit, or share any personal data or user information. No analytics, no tracking, no external servers.

## How It Works

When you click Export, the extension reads the chat content directly from the page you are currently viewing and immediately downloads it as a file to your device. Everything happens locally in your browser.

## Permissions

- **activeTab / scripting / host permissions** — used solely to read the chat content on the current page in order to generate the export file.
- **downloads** — used solely to save the exported file to your device.

No permission is used to collect or transmit data.

## Third Parties

No data is shared with any third party. The extension makes no network requests.
