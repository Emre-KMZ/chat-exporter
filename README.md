# Chat Exporter

A browser extension that lets you export conversations from AI chat platforms as Markdown files.

## Supported Platforms

- [Claude](https://claude.ai)
- [ChatGPT](https://chat.openai.com) / [chatgpt.com](https://chatgpt.com)
- [Gemini](https://gemini.google.com)

## How It Works

When you are in a chat and click the extension popup, then hit export, it extracts the current conversation from the page and downloads it as a `.md` file.

## Installation

1. Clone this repo
2. Open your browser's extension management page (e.g. `chrome://extensions`)
3. Enable **Developer mode**
4. Click **Load unpacked** and select this folder

## Usage

1. Open a conversation on Claude, ChatGPT, or Gemini
2. Click the **Chat Exporter** extension icon
3. Choose your export format and click export
4. The conversation is downloaded as a Markdown file
